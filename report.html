Instructions

The compiler comes in the form a python 3 file run_compiler.py.

which takes one argument that specifies the location of the C source file:

python3 run_compiler.py <path_to_c_file>

We also have a set of test scripts located in the folder test_scripts. They have to be run from the same directory as run_compiler.py. For example:

./test_scripts/test_array.sh 

Once the file has been run it will produce two dot-files in the output folder. One displays the AST-tree, and one displays the Symbol Table. LLVM code will not yet be outputted.
