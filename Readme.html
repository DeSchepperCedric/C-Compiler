Optional Features

Implemented features

  • %, +=, -=, *=, /=, <=, >=, != (fully implemented)
  • Conversions in assignments, parameter passing, expressions, etc (automatic + casts) (fully implemented)
  • Pruning of dead code for 'continue', 'return', 'break' (fully implemented).
  • Bool type, 'not', 'and', 'or', 'true', 'false' keywords (in grammar)(fully implemented).
  • Bool type used as result of boolean expressions: comparisons, logic, etc (fully implemented).
  • ++, --, prefix and postfix (fully implemented)
  • Unary +,-
  • Constant folding
  • Constant propagation
  • for loop

Implementation Status

All mandatory features are implemented:
  • Grammar
  • AST
  • LLVM
  • MIPS
  • Types: pointer, void, char, int, float
  • Import, printf, scanf
  • If, else, return, while
  • Local variables

  • Comments
  • Operations: =, +, -, *, /, <, >, ==
  • Functions, function calling
  • Array
  • Syntax errors
  • Typing errors
  • Pruning after 'return'

Instructions

The compiler comes in the form of a python 3 file run_compiler.py. It can be used as follows:

python3 run_compiler.py <language> <path_to_c_file>

The language can either be mips or llvm. This will create an output folder that contains a dot representation of the AST-tree and the symbol table. It will also generate a .ll or .asm file for respectively llvm and mips.
