Optional Features

  • for (not yet implemented in LLVM, AST)
  • break, continue (not yet implemented in LLVM)
  • Unary +,- (not yet implemented in LLVM)
  • &&, ||, ! (not yet implemented in LLVM)
  • %, +=, -=, *=, /=, <=, >=, != (fully implemented)
  • Return check
  • Conversions in assignments, parameter passing, expressions, etc (automatic + casts) (fully implemented)
  • Constant folding (not yet implemented in AST)
  • Pruning of dead code for 'continue', 'return', 'break' (fully implemented).
  • Bool type, 'not', 'and', 'or', 'true', 'false' keywords (fully implemented).
  • Bool type used as result of boolean expressions: comparisons, logic, etc (fully implemented).

Implementation Status

All mandatory features are implemented:
  • Grammar
  • AST
  • LLVM
  • Types: pointer, void, char, int, float
  • Import, printf, scanf
  • If, else, return while
  • Local variables
  • Global variables
  • Comments
  • Operations: =, +, -, *, /, <, >, ==
  • Functions, function calling
  • Array
  • Syntax errors
  • Typing errors
  • Pruning after 'return'

Instructions

The compiler comes in the form of a python 3 file run_compiler.py. It can be used as follows:

python3 run_compiler.py <path_to_c_file>

This will create an output folder that contains a dot representation of the AST-tree and the symbol table, and an .ll file that contains the LLVM code.
